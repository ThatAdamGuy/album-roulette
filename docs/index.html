🎲 Album Roulette

One click plays a random album from your YouTube Music library — start to finish, the way albums were meant to be heard.

No streaming service offers this natively.

🎲 Fully random

A 🎲 button appears in the YouTube Music header — plus a toolbar icon and keyboard shortcut Alt+Shift+R that work from any tab. Click it and a random album from your library starts at track 1, in order, no shuffle. It avoids repeating your last 20 picks, and it includes your uploaded albums.

Genre picks

Chips like Jazz · 74 play a random album within that genre. Genres are identified automatically in the background using free public music databases (Apple's iTunes Search and MusicBrainz) — a one-time lookup per album, cached forever, no account or API key required.

Semi-🎲 "Deal me 6"

Shows six album covers from six different genres at once, so you can pick by mood instead of leaving it fully to chance.

Keep Rolling 🔁 (optional)

When an album ends, a gentle four-note chime marks the boundary, then another random album starts automatically. Off by default — flip it on whenever you want the music to keep going.

Seasonal smarts

Holiday albums sit out of full-random rolls outside the season, so a Christmas record won't ambush you in July. They stay available any time via their genre chip.

🔒 Private by design

No account, no API key, no server, no analytics. Your library list stays in your browser. The only thing ever sent anywhere is an album's title + artist, to public genre databases — nothing else.

Install

Coming to the Chrome Web Store
Install manually in the meantime
  1. Download or clone the GitHub repo: github.com/ThatAdamGuy/album-roulette
  2. Open chrome://extensions in Chrome
  3. Turn on Developer mode (top right)
  4. Click Load unpacked and select the folder you downloaded
  5. Open music.youtube.com, signed in, and look for the 🎲
